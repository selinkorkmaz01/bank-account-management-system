# Bank Account Management System

A simple Java-based Bank Account Management System developed using Object-Oriented Programming (OOP) principles.

## Features

- Create bank accounts
- Deposit money
- Withdraw money
- Display account information
- Savings account with interest calculation
- Demonstration of inheritance and encapsulation

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Project Structure

```text
Account.java
SavingsAccount.java
Main.java
```

## OOP Concepts Demonstrated

- Encapsulation
- Inheritance
- Constructors
- Method Reuse
- Class Relationships

## Purpose

This project was developed for academic learning and practice of Java programming and Object-Oriented Programming concepts.
